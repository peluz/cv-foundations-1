Estrutura do projeto:
	Arquivo proj1.py com o codigo em python
	Diretório data com as imagens default

Requisitos:
	Python (Versão usada foi a 3.5.2)
	OpenCV (Versão usada foi a 3.3.0)

Como rodar:
	A partir do diretório raiz, rodar "python proj1.py [número do requisito]",
	onde [número do requisito] corresponde a 1, 2, 3 ou 4 dependendo do requisito a
	ser testado.

	Para testar em imagem/vídeo diferente dos default, acrescentar ao comando "-file caminho/para/imagem"
	Por exemplo:
	python proj1.py 2 data/lena.png
